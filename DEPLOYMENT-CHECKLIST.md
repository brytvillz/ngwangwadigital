## ✅ Technical Setup

### SEO & Meta Tags

- [ ] Verify all page titles are unique
- [ ] Check all meta descriptions are under 160 characters
- [ ] Add Open Graph image URLs (after deployment)
- [ ] Update og:url to your actual domain
- [ ] Test meta tags with [Meta Tags](https://metatags.io/)

### Forms

- [ ] Verify Netlify form attributes are present
- [ ] Test contact form after deployment
- [ ] Set up email notifications in Netlify dashboard
- [ ] Test waitlist form (products page)

### Links

- [ ] Test all internal links work
- [ ] Verify all external links have `target="_blank"` and `rel="noopener"`
- [ ] Check WhatsApp link works on mobile
- [ ] Test Instagram link
- [ ] Verify demo link to Virtual Waitress

### Mobile Responsiveness

- [ ] Test on iPhone (Safari)
- [ ] Test on Android (Chrome)
- [ ] Test tablet view (iPad)
- [ ] Verify hamburger menu works
- [ ] Check touch targets are large enough
- [ ] Test form inputs on mobile

### Performance

- [ ] Compress all images (use TinyPNG.com)
- [ ] Test site speed on [PageSpeed Insights](https://pagespeed.web.dev/)
- [ ] Aim for 90+ Lighthouse score
- [ ] Test on slow 3G connection
- [ ] Verify lazy loading works

## ✅ Deployment

### Pre-Deployment

- [ ] Test all pages locally
- [ ] Check browser console for errors
- [ ] Verify no broken links
- [ ] Test forms locally (won't submit but should validate)

### Netlify Deployment

- [ ] Create Netlify account
- [ ] Deploy site (drag & drop or Git)
- [ ] Verify all pages load correctly
- [ ] Test contact form submission
- [ ] Check form submissions in Netlify dashboard
- [ ] Set up form notifications (email)

### Domain Setup

- [ ] Purchase domain (if not done)
- [ ] Connect custom domain in Netlify
- [ ] Update DNS records with domain provider
- [ ] Wait for DNS propagation (can take 24-48 hours)
- [ ] Verify SSL certificate is active (HTTPS)
- [ ] Test www and non-www versions redirect correctly
